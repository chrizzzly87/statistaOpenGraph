How to use the service:

Use one of the following routes to create a request that will render and export an image

Job-Position

Route

/og/job

Example: http://localhost:1987/og/job?text=Team%20Webdev%20ist%20das%20Beste%20Team%20bei%20Statista&department=it&location=newyork&platform=en

Params

text

Required This is the main text that will be drawn. Make sure to urlencode the text.

id

Required Used for the unique filename

deparment

Currently not used. Could be added to render the department name OR define the bg image based on the department

location

Render the location of the job position. Currently these locations are available:

  • New York
  • Hamburg
  • Singapure
  • London
  • Tokyo
Default: Worldwide

Platform

Currently not used. Might be useful for translations. Default: en
